粤式 “通关秘籍” 已到手！从零开始闯荡粤语江湖

最近看了由马丽主演的《水饺皇后》，因为故事发生在香港，里面使用粤语非常频繁，就发现讲粤语非常有意思，然后又想起去香港的经历，所以我决定从零基础开始学习粤语。

常见问候语

粤语 释义 粤拼 谐音
早晨 早上好 zou2 san4 走森
唔该 谢谢/劳驾 m4 goi1 唔该
你好 你好 nei5 hou2

人称

粤语 释义 粤拼 谐音
我 / 我們 我 / 我们 ngo5 / ngo5 dei6 我 / 我哋
你 / 你們 你 / 你们 nei5 / nei5 dei6 唔该
他 / 他們 他 / 他们 keoi5 / keoi5 dei6 佢 / 佢哋

方位词

粤语 释义 粤拼 谐音
边度 哪里 bin1 dou6 宾豆
呢度 这里 nei1 dou6 内豆
嗰度 那里 go2 dou6 果豆
边個 哪个 bin1 go3 宾果
呢個 这个 nei1 go3 内果
嗰個 那个 go2 go3 果果
边位 哪位 bin1 wai6 宾歪
呢位 这位 nei1 wai6 内歪
嗰位 那位 go2 wai6 果歪
